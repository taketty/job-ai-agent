# job-ai-agent

> 就活の全工程をAIが代行する、次世代の全自動就活エージェントOS

## 概要

**job-ai-agent** は、求職者が最初のチャットを終えるだけで、あとはAIが面接・日程調整・内定選択まで全工程を代行する全自動就活・転職代行プラットフォームです。

コンプライアンス強化により「本音」を探り合えなくなった就活市場において、AIが人間のバイアスや過度な緊張を排除し、純粋なデータ照合による本質的なマッチングを実現します。

Amazon Bedrock のマルチエージェントオーケストレーションと AWS サーバレスアーキテクチャを基盤とし、Google Calendar・Gmail と連携することで、就活の全プロセスを自動化します。

---

## 主な機能

| 機能 | 説明 |
|------|------|
| 初期ヒアリング（壁打ちAI） | チャットでユーザーの希望条件・経歴・「本当の欲望」を収集し初期パラメータを生成 |
| ES 自動生成 | 初期パラメータをもとに企業ごとに最適化されたエントリーシートを自動生成 |
| カレンダー自動調整 | Google Calendar と連携し、面接日程を企業側と自動調整・登録 |
| 面接 AI 代行 | 求職者AI と企業AI が双方向で面接を実施し、聞きたいことを自動集約 |
| 内定レコメンド | 複数内定からユーザーの初期パラメータに最もマッチする1社をAIが推薦 |
| 辞退メール自動生成 | Gmail API で辞退メールを自動生成・送信 |

---

## アーキテクチャ概要

```
[AWS Amplify（Next.js ホスティング）]
       ↓ HTTPS
[API Gateway（REST / WebSocket）]
       ↓
[Lambda + Step Functions]     ←→  [Amazon Bedrock Agents]
       ↓                                    ↓
[DynamoDB / S3]              ←→  [Google Calendar / Gmail API]
```

- **クラウド**: AWS（完全サーバレス構成）
- **AI 基盤**: Amazon Bedrock（Claude）+ Bedrock Agents マルチエージェント
- **フロントエンド**: Next.js（AWS Amplify ホスティング・CDN）
- **外部連携**: Google Calendar API / Gmail API
- **認証**: Amazon Cognito
